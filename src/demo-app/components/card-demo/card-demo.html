Cards

The MDC Card component is a sheet of material that serves as an entry point to more detailed information.
Directive
mdc-card
Directive
mdc-card-horizontal Stack multiple card blocks horizontally instead of vertically, by placing them inside of a horizontal block.
mdc-card-supporting-text Used for displaying the bulk of the textual content of the card.
mdc-card-media Used for showing rich media in cards, and optionally as a container.
Directive
mdc-card-primary Defines the primary text / title content block.
Child Directive
mdc-card-title A title block.
Properties Description
@Input() large: boolean An option to make the title text larger. (default is true)
Child Directive
mdc-card-subtitle A subtitle block.
Directive
mdc-card-media-item Media items are designed to be used in horizontal blocks, taking up a fixed height, rather than stretching to the width of the card.
Properties Description
@Input() size: number Some predefined media item sizes. Valid values are [1, 1.5, 2, and 3]
Directive
mdc-card-actions Place mdc-card-button's inside the action content.
Properties Description
@Input() vertical: boolean Option to lay actions out vertically instead of horizontally.
Directive Description
button[mdc-card-button]
a[mdc-card-button] Compact and spec-aligned buttons for card actions.
