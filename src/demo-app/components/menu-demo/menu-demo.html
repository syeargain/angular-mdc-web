Menu

The MDC Menu component is a spec-aligned menu component adhering to the Material Design menu requirements.

API reference

Component
mdc-menu
Properties Description
openFrom: string Override the opening point of the menu. (default: topLeft)
Methods Parameters
open(focusIndex?: number)
isOpen(): boolean Returns whether or not the menu is open.
close() Closes the menu.
isFocused(): boolean Returns whether or not the menu has focus.
focus() Set focus to the menu element.
Event Description
cancel() Dispatches an event that the menu closed with no selection made. 
select(item: HTMLElement, index: number)
 Dispatches an event that a menu item has been selected.
Directives
mdc-menu-anchor Use to anchor a menu to an element.
mdc-menu-item Single list item inside of a menu.
mdc-menu-divider Draws a line at the bottom of the menu item.
