Select Menus

MDC Select provides Material Design single-option and multi-option select menus.
Component
mdc-select
Properties Description
id: string Unique Id of the select (auto generated if not supplied).
name: string Name of the select.
label: string Label text to be shown if no value has been selected.
value: string Value of the checkbox.
tabIndex: number Set the underlying tab index of the select. (Default is 0)
closeOnScroll: boolean Closes the menu on window scroll. (Default is true)
Methods Parameters
focus() Sets focus to the select component.
getValue(): string Returns the value of the currently selected option, or an empty string if no option is selected.
open(index?: number = 0) Opens the select menu and sets focus to provided index.
close() Closes the select menu.
isOpen(): boolean Returns whether or not the menu is open.
getSelectedIndex(): number Returns the index of the currently selected option. Returns -1 if no option is currently selected.
setSelectedIndex(selectedIndex: number) Sets the selected index of the component.
isDisabled(): boolean Returns whether or not the select is disabled.
setDisabled(disabled: boolean) Enables/disables the select.
Events Description
change(index: number, value: string) Event dispatched on value change.
Directive
mdc-select-item
Properties Description
value: string Value of the select item.
disabled: boolean Disable selection of the select item.
